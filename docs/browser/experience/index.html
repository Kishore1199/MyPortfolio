Mysales

    MySales is a CRM (Customer Relationship Management) application designed to help businesses, manage their interactions with customers and potential customers. It provides a centralized platform for storing and organizing customer data, tracking sales activities, and facilitating communication between sales teams and customers. MySales provides the exhaustive report and tracking facilities to reviewers to identify in and out transactions to the organization. JWT ensured secure authentication. Separate MySQL databases provided data isolation and scalability. Eureka Server and Spring Cloud API Gateway facilitated service discovery and routing. Log4j2 and Splunk enabled real-time monitoring. Key contributions included migrating monolithic applications to microservices, developing backend APIs, optimizing code quality with SonarQube.

    Using Technologies :

    Microservices, Java , Spring Boot 3.x, Angular 17, Spring Cloud API gateway 2022.0.2, Spring Security, JWT , Maven, Hibernate 6.x, MySQL.
